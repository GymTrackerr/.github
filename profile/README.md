## GymTracker's Goal
GymTracker is a fitness tracking application that lets users log their workout sessions, and track their progress over time. Users can choose for the app to be local only, or sync their data to the cloud so they can access it from multiple devices, and gain access to a web dashboard.

## Current Status
GymTracker is currently in active development. The mobile app is being developed for iOS using Swift and SwiftUI. The backend is being developed using NestJS with TypeScript. The web dashboard is being developed using Svelte with TypeScript.
